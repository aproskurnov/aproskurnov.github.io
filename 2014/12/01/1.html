Homestead 2.1 и возникшие проблемы в ubuntu 14.04

После установки homestead 2.1 процесс которой прекрасно описан в официальной документации laravel возникли 2 проблемы про которые там не упомянули. Во-первых, нужно создать новую папку для синхронизации гостевой и хостовой системой. По умолчанию это 

~/Code
Во-вторых, при выполнении команды 
homestead edit
для редактирования конфига, на ubuntu 14.04 вылезла ошибка 
couldn't get a file descriptor referring to console
, это легко поправить. Для этого открываем файл 
~/.composer/vendor/laravel/homestead/src/EditCommand.php
И заменяем 47 строчку 
return strpos(strtoupper(PHP_OS), 'WIN') === 0 ? 'start' : 'open';
на 
return 'xdg-open';
Все, теперь при выполнении homestead edit в редакторе открывается конфиг Homestead.yaml
